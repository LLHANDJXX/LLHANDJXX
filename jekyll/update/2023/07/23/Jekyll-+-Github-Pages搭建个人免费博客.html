Windows 上安装 Jekyll

Jul 23, 2023

虽然 Windows 不是官方支持平台，它可以在适当调整后运行 Jekyll。

安装 Ruby 和 Jekyll

通过 RubyInstaller 安装

安装 Ruby 和 Jekyll 最简单的方式是用 Windows 版 RubyInstaller。

RubyInstaller 是一个自包含的 Windows 安装程序，它包含 Ruby 语言、一个执行环境、重要文档、及其它。

这里只讨论 RubyInstaller-2.4 及以后的版本。老的版本需要手动安装 Devkit。

  1. 从 RubyIntaller Downloads 下载并安装一个 Ruby+Devkit 版。使用默认选项安装。

  2. 在安装向导的最后阶段运行 ridk install 步骤。这是安装带有 native extensions 的 gems 所必需的。你可以在 RubyInstaller 文档中找到更多有关的信息。从选项中选择

    MSYS2 and MINGW development tool chain

  3. 从开始菜单打开一个新的 cmd 窗口，以便 PATH 环境变量的更改生效。使用

    gem install jekyll bundler

    安装 Jekyll 和 Bundler 。

  4. 检查 Jekyll 是否安装成功

    jekyll -v

在检查Jekyll 是否未正确安装时，您可能会收到错误。重新启动操作系统并再次运行 jekyll -v 。 如果错误仍然存在，请打开 RubyInstaller 问题.

快速开始

  1. 通过 cmd 进入想安装的位置，运行

    jekyll new myblog

  2. 进入目录

    cd myblog

  3. 运行服务

    bundle exec jekyll serve

  4. 浏览 http://localhost:4000

如果你使用 Ruby version 3.0.0 或更高, 第3步 可能出错。 你可能需要通过 bundle add webrick 在依赖中添加 webrick 来修复。

传 --livereload 选项给 serve 以便在修改源文件时自动刷新页面: bundle exec jekyll serve --livereload
